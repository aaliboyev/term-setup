# term-setup

Portable terminal environment: Ghostty + zsh (starship, fzf, fzf-tab,
autosuggestions, syntax highlighting, eza, yazi).

Requires [Homebrew](https://brew.sh). Appends to `~/.zshrc`, writes
`~/.config/ghostty/config`.

```sh
curl -fsSL https://raw.githubusercontent.com/aaliboyev/term-setup/main/setup.sh | bash
```
